# Temperature Converter

A simple temperature conversion web application built with HTML, CSS, and JavaScript. The app allows users to convert temperatures between Celsius and Fahrenheit through an easy-to-use interface.

## Features

* Convert Celsius to Fahrenheit
* Convert Fahrenheit to Celsius
* Input validation for empty fields
* User-friendly error and hint messages
* Clean and responsive design
* Hover effects and modern styling

## Technologies Used

* HTML5
* CSS3
* JavaScript (Vanilla JS)

## How It Works

1. Enter a temperature value in the input field.
2. Select the desired conversion type:

   * Celsius → Fahrenheit
   * Fahrenheit → Celsius
3. Click the **Submit** button.
4. View the converted temperature instantly.

## Validation

The application handles common user errors:

* Displays an error message when the input field is empty.
* Displays a hint when no conversion option is selected.
